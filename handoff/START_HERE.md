# Edit Studio 디자이너 전달 패키지

이 폴더는 Edit Studio를 디자이너가 로컬에서 실행하고 테스트할 수 있도록 정리한 전달용 패키지입니다.

## 가장 빠른 실행 방법

1. 압축 파일을 풉니다.
2. 터미널을 엽니다.
3. 압축을 푼 폴더로 이동합니다.

가장 쉬운 방법은 터미널에 `cd `를 입력한 뒤, 압축을 푼 폴더를 Finder에서 터미널 창으로 드래그 앤 드롭하고 Enter를 누르는 것입니다.

예시:

```bash
cd /Users/내이름/Downloads/Edit-Studio-designer-handoff
```

터미널 경로가 앱 폴더 안으로 이동했는지 확인하려면 아래 명령을 입력합니다.

```bash
ls
```

`package.json`, `src`, `dist`, `START_HERE.md` 같은 파일이 보이면 올바른 위치입니다.

4. 아래 명령을 한 번 실행합니다.

```bash
npm install
```

5. 개발 서버를 실행합니다.

```bash
npm run dev
```

6. 브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:5173/
```

## 자세한 설명서

자세한 실행 방법과 사용법은 `Edit-Studio-Designer-Guide.md` 파일을 확인하세요.

## 꼭 알아둘 점

- 영상 파일은 서버로 업로드되지 않고 브라우저 안에서만 처리됩니다.
- 편집 내용은 브라우저에 자동 저장되며, 같은 브라우저에서는 원본 영상도 자동 복구를 시도합니다.
- 자동 복구가 안 되거나 다른 컴퓨터에서 열 때는 화면의 `다시 연결` 버튼으로 같은 원본 영상을 선택하면 이어서 작업할 수 있습니다.
- MP4 저장은 브라우저에서 FFmpeg WASM을 실행하므로 첫 실행과 긴 영상 렌더링은 시간이 걸릴 수 있습니다.
